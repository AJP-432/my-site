# My Personal Site

Welcome to my personal site and static blog built with Svelte.

## Getting Started

### Creating a Project

If you need to create a new Svelte project, you can use the following command:

```bash
npx sv create my-app
```

### Installing Dependencies

Install the necessary dependencies:

```bash
npm install
```

## Development

Start the development server:

```bash
npm run dev
```

To start the server and open the app in a new browser tab:

```bash
npm run dev -- --open
```

## Building

Create a production build of your app:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

> Note: To deploy your app, you may need to install an [adapter](https://svelte.dev/docs/kit/adapters) for your target environment.
